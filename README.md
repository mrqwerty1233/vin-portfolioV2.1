# Personal Portfolio Website

This is my personal developer portfolio built using **React + Vite + Framer Motion**.  
It showcases my skills, projects, and professional background as an IT graduate specializing in web development, IT support, and systems solutions.

This project is still **in progress** and will continue to be improved over time.

# Running the Project

### 1 Install Node.js

Download from:
https://nodejs.org

Recommended version:

Node 18+


Check installation:


node -v


---

### 2 Clone the Repository


git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git


Then open the folder:


cd YOUR_REPOSITORY


---

### 3 Install Dependencies


npm install


This installs all packages listed in `package.json`.

---

### 4 Start Development Server


npm run dev


The site will run at:


http://localhost:5173


---

# Working on Multiple Computers

Since I use **two computers (personal PC and work PC)**, I use GitHub to keep the project synchronized.

Workflow:

### On Personal PC

1 Push changes to GitHub


git add .
git commit -m "update portfolio"
git push


---

### On Work PC

Clone the project:


git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git


Install dependencies:


npm install


Run the project:


npm run dev


---

### When switching between PCs

Always pull the latest version:


git pull
